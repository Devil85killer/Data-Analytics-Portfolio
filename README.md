## 📌 Project Overview
This project focuses on analyzing the sales performance and customer retention (churn) of an e-commerce platform. The goal was to transform raw transactional data into actionable business intelligence, helping stakeholders understand purchasing trends, regional performance, and why customers stop buying.

![E-Commerce Dashboard Preview](Screenshot%202026-03-30%20051856.png)

## 🎯 Business Problem Solved
* **Sales Tracking:** Monitored total revenue, average order value (AOV), and daily sales trends.
* **Customer Retention:** Identified customer churn patterns using advanced SQL queries to segment active vs. lost customers.
* **Geographical Insights:** Mapped out top-performing regions to optimize marketing and delivery operations.

## 🛠️ Tools & Tech Stack
* **Data Extraction & Manipulation:** SQL (`CHURN ANALYSIS.sql`)
* **Data Visualization & Modeling:** Power BI (`Olist E-Commerce Sales Dashboard.pbix`)
* **Key Functions Used:** DAX, Data Modeling (Star Schema), SQL `JOIN`s, Aggregations, and Subqueries.

## 📂 Project Files
* `CHURN ANALYSIS.sql`: Contains the complex queries written to calculate churn rates and customer lifetime metrics.
* `Olist E-Commerce Sales Dashboard.pbix`: The complete, interactive Power BI report with dynamic slicers and custom visuals.
